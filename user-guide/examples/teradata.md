# Teradata table DDL Example

## Varchar Table
```sql
CREATE MULTISET TABLE listener.test_varchar ,NO FALLBACK ,
     NO BEFORE JOURNAL,
     NO AFTER JOURNAL,
     CHECKSUM = DEFAULT,
     DEFAULT MERGEBLOCKRATIO
     (
      uuid VARCHAR(55) CHARACTER SET LATIN CASESPECIFIC,
      source_id VARCHAR(55) CHARACTER SET LATIN CASESPECIFIC,
      time_stamp TIMESTAMP(6) WITH TIME ZONE,
      data VARCHAR(64000) CHARACTER SET LATIN NOT CASESPECIFIC)
PRIMARY INDEX ( uuid );
```
